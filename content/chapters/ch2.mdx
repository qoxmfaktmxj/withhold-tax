- **본법 개정**: 1월 1일 개정 발효
- **시행령 개정**: 2월 10일 전후
- **시행규칙 개정**: 3월 10일 전후 (서식 포함)

**조문 읽는 법**: `소득세법 제4조 제1항 제1호 가목` · 구조: 조 → 항 → 호 → 목 · 시행령은 본법을 이길 수 없음(상위법 우선).

<Box kind="tip" title="◆ Pro Tip · 법령 검색 사이트">
**국가법령정보센터**(law.go.kr) — 모든 공포 법령 게재. **법제처 현행법령**은 실무 작업 시(링크 잘 걸림), **최신법령**은 개정 이력 확인 시. **국민참여입법센터**는 입법 예고 단계 확인용(서식 미리보기). 법령정보 API(open.law.go.kr)를 AI에 연동하면 최신 법령 검증 가능.
</Box>

## 수정신고 vs 경정청구 ★

<Compare
  left={
    <>
      <strong>수정신고 (증액)</strong>
      <ul>
        <li>방향: <strong>세액 ↑</strong> (더 내는 것)</li>
        <li>효력: <strong>자동 경정</strong></li>
        <li>결정주체: 납세자 본인</li>
        <li>국가 손해: 없음 (자동 인정)</li>
      </ul>
    </>
  }
  right={
    <>
      <strong>경정청구 (감액)</strong>
      <ul>
        <li>방향: <strong>세액 ↓</strong> (돌려받는 것)</li>
        <li>효력: <strong>청구 효력만</strong></li>
        <li>결정주체: <strong>세무서</strong>가 결정</li>
        <li>국가 손해: 있음 (검토 필요)</li>
      </ul>
    </>
  }
/>

<Box kind="warn" title="◆ 중요 용어">
**경정기관 = 세무서** · **기한 후 신고**: 신고기한 지난 후 신고 → 결정력 없음(세무서가 결정). 4대보험도 동일 용어 사용.
</Box>
