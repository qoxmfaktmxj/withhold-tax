1. 수치·기한·가산세·세율 항목에 `as-of date`와 공식 출처를 붙이기
2. 2026년 개정·시행 항목을 별도 장으로 분리하기
3. 비거주자·외국법인 원천징수, 조세조약, 이자·배당 원천징수 장을 보강하기
4. 강사 코멘트·실무 경험담·법령 확정 사항을 시각적으로 분리하기

현재 HTML은 해외파견·거주자 판정은 비교적 풍부하지만, inbound 비거주자·외국법인 원천징수 실무가 약합니다. 별도 Chapter로 추가하는 것이 좋습니다.

권장 소목차:

1. 거주자·비거주자·내국법인·외국법인 판정
2. 국내원천소득 유형별 판정 기준
3. 조세조약 적용 순서
4. 제한세율·면제신청서·거주자증명서·수익적 소유자 확인
5. 국내사업장 귀속 여부
6. 사용료·인적용역·기술용역·이자·배당 사례

### 3.3 이자·배당 원천징수 장

2. 비영업대금 이익 25%

6. 고배당기업 과세특례
7. 비거주자·외국법인 이자·배당 제한세율

## 4. HTML 구조 개선안

### 4.1 출처 배지

각 핵심 문장 옆에 다음 배지를 붙입니다.

- `NTS`: 국세청 공식 안내
- `LAW`: 국가법령정보센터 법령 조문
- `BOOK`: CFO Academy 교재
- `LECTURE`: 강의 녹음 기반 설명
- `CASE`: 강사 사례 또는 실무 사례
- `CHECK`: 재검증 필요

### 4.2 데이터 속성

핵심 수치에는 `data-asof`, `data-source`, `data-law`, `data-risk`를 붙입니다.

예시:

```html
<span class="fact" data-asof="2026-06-08" data-source="NTS" data-law="국세청 원천징수 개요" data-risk="high">
  인적용역 사업소득은 2024.7.1 이후 지급분부터 1,000원 미만이어도 원천징수
</span>
```

### 4.3 검증 상태 테이블

HTML 상단에 다음 표를 추가합니다.

| 상태 | 의미 |
|---|---|
| VERIFIED | 공식 출처와 대조 완료 |
| NEEDS-LAW | 법령 조문 확인 필요 |
| NEEDS-FORM | 서식·홈택스 화면 확인 필요 |
| LECTURE-ONLY | 강의 내용 기반, 공식 출처 미확인 |
| CASE-ONLY | 사례 설명, 일반화 금지 |
