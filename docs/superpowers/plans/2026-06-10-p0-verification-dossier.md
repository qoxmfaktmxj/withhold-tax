## 공통 주의
- API DRF 링크에 OC키 포함 → 산출물 lawUrl엔 공개형 `law.go.kr/법령/...`만 사용.
- 적용시기는 부칙 적용례 직접 확인 전까지 scopeLimitations에 "적용시기=세무사회 자료 기준" 명시, confidenceScore 85~90.
