## 1. 빌드 방식 (재현 가능한 파이프라인)

### 1-1. 디렉토리

### 1-2. 조각 구성 (연결 순서 중요)
```

hero.html      # 히어로 섹션 + 목차(TOC) 카드
```

### 1-3. 연결 명령
```bash
cd /home/claude/build && \
cat head.html hero.html day1_a.html day1_b.html day2_a.html day2_b.html tail.html \
  > /home/claude/원천징수실무_완전정리_0521-0522.html
```

### 1-4. 검증 파이프라인 (전달 전 필수)
1. **태그 균형 체크** — Python `re.findall` 로 element별 open/close 카운트 비교
   ```python
   import re
   html = open('원천징수실무_완전정리_0521-0522.html', encoding='utf-8').read()
   for tag in ['section','div','table','header','footer','nav','script','style']:
       o = len(re.findall(r'<'+tag+r'[ >]', html)); c = len(re.findall(r'</'+tag+r'>', html))
       print(tag, o, c, 'OK' if o==c else '!!! MISMATCH')
   # DOCTYPE/</html>/</body> 각 1개, 섹션 ID 셋 확인
   ```
2. **Playwright 시각 검증** — `scrollIntoView()` 로 주요 섹션(hero/toc/ch1/cheatsheet) 스크린샷
   ```bash
   pip install playwright --break-system-packages -q
   playwright install chromium
   # page.evaluate("document.getElementById('ch1').scrollIntoView()") 후 screenshot
   ```
3. 소소한 수정 → `str_replace` → 최종본 `/mnt/user-data/outputs/` 복사 → `present_files`

## 1. 소득세 개요

### 소득세의 정의
- 소득을 과세대상으로 부과하는 조세
- **개인소득세** → 소득세법상 "소득세"
- **법인소득세** → 소득세법상 "법인세"
- 통상 "소득세"라 하면 개인소득세를 의미

### 주요 용어 (소득세법)
| 구분 | 정의 |
|------|------|
| **거주자** | 국내 주소를 두거나 183일 이상 거소를 둔 개인 |
| **비거주자** | 거주자가 아닌 개인 |
| **내국법인** | 국내 본점/주사무소 또는 **사업의 실질적 관리장소**를 둔 법인 |
| **외국법인** | 내국법인이 아닌 법인 |

> **포인트**: 거주자/비거주자에 따라 과세범위가 달라짐. 183일(6개월)이 상징적 기준.

### "사업의 실질적 관리장소" 규정 배경
- **론스타-스타타워 사건**이 계기
- 론스타가 스타타워 법인 지분을 싱가포르 투자청(GIC)에 매각 → 약 8천억 양도차익
- 의사결정 장소를 벨기에라 주장 → 한국 과세권 회피 시도
- 이후 실질적 관리장소가 국내면 내국법인으로 보아 과세하도록 규정 신설

## 2. 소득의 과세방식: 열거주의 vs 포괄주의

| 구분 | 적용 법 | 내용 |
|------|--------|------|
| **소득원천설(열거주의)** | 소득세법 | 법에 규정된 소득만 과세 |
| **순자산증가설(포괄주의)** | 법인세법 | 합법/불법 불문, 소득 발생 시 과세 |

- **개인의 코인·주식 소득** → 현재 열거 안 됨 → 비과세 (코인은 유예 중)
- **법인의 코인·주식 소득** → 포괄주의로 과세
- **유형별 포괄주의**: 이자·배당·사업·근로·연금소득은 제한적 포괄주의 적용 (유사 신상품 과세 가능)

## 3. 소득세법상 8가지 소득

```
종합소득 (6가지)              분류과세 (2가지)
├─ 이자소득                   ├─ 퇴직소득
├─ 배당소득                   └─ 양도소득
├─ 사업소득
├─ 근로소득
├─ 연금소득
└─ 기타소득
```

### 종합과세 이유
- 누진세 구조 → 합산 시 세부담 증가
- 최고세율 45% + 지방소득세 4.5% = **49.5%**
- 과세표준 1.5억 구간에서 세율 24% → 35% (11%p 점프)

### 분류과세 이유
| 소득 | 분류과세 사유 |
|------|--------------|
| **퇴직소득** | 장기 누적 소득 → 연분연승법 적용 (근속연수로 나눔) |
| **양도소득** | 장기 보유 자산 차익 + 중과세율 별도 적용 (다주택자 등) |

## 4. 원천징수 대상

| 소득 | 원천징수 |
|------|---------|
| 이자소득 | ⭕ |
| 배당소득 | ⭕ |
| 사업소득 | △ (일부 3.3%) |
| 근로소득 | ⭕ |
| 연금소득 | ⭕ |
| 기타소득 | ⭕ |
| 퇴직소득 | ⭕ |

> 원천징수는 **세금 떼먹는 것 방지** 목적. 거의 모든 소득에 적용되며, 개인과 강하게 연결.
> 법인에게 돈 줄 때는 원칙적으로 원천징수 X (이자소득 등 예외)

## 5. 소득금액 계산 구조

### 회계 vs 세법 용어
| 회계 | 소득세법 |
|------|---------|
| 수익 | 총수입금액 |
| 비용 | 필요경비 |
| 이익 | 소득금액 |

**소득금액 = 총수입금액 − 필요경비**

### 소득별 계산 구조
| 소득 | 총수입금액 | 필요경비 |
|------|-----------|---------|
| 이자·배당 | ⭕ | ❌ (인정 안 함) |
| 사업·기타 | ⭕ | ⭕ |
| 근로 | 총급여 | 근로소득공제 |
| 연금 | 총연금액 | 연금소득공제 |

> 4대보험은 일반적으로 **총수입금액(총급여) 기준** 산정

## 6. 종합소득세 신고 특징

1. **과세기간**: 1.1 ~ 12.31 (엄격, 변경 불가)
2. **개인별 과세** (단, 주택 수 판단 시 세대합산)
3. **누진세율 적용**
4. **인적공제** 적용 (부양가족)
5. **신고납세제도** (본인이 신고)
6. **주소지 관할 세무서** 신고
7. **원천징수제도** 병행

## 7. 법체계 (3단 입법구조)

```
소득세법(본법)     ← 국회 제정 (법률)
   ↓ 위임
시행령            ← 정부 제정 (대통령령)
   ↓ 위임
시행규칙          ← 기획재정부장관 (부령)
```

### 개정 시기
- **본법**: 1월 1일
- **시행령**: 2월 10일 전후
- **시행규칙**: 3월 10일 전후 (서식 포함)

### 조문 읽는 법
- 例) 소득세법 제4조 제1항 제1호 가목
- 구조: **조 → 항 → 호 → 목**
- 항: ①②③ (동그라미 숫자)
- 호: 1, 2, 3
- 목: 가, 나, 다

> 시행령은 본법을 이길 수 없음 (상위법 우선)

## 9. 기부금과 적격증빙

### 기부금은 적격증빙 의무 없음
- 적격증빙 4가지: 세금계산서, 계산서, 신용카드 매출전표, 현금영수증
- **추가**: 기부금영수증도 포함되어야 함
- 3만원 초과 거래에서 적격증빙 미수취 시 **증빙불비가산세 2%**
- **단, 기부금은 일방적 지급(대가 X)이므로 가산세 대상 아님**

> 적격증빙 수취 요건: 거래상대방이 사업자 + 재화·용역 공급 대가
> → 직원에게 주는 돈은 사업자가 아니므로 적격증빙 대상 X

## 10. 임금 vs 보수 (노동법 vs 세법)

### 용어 구분
| 구분 | 노동법 | 세법 |
|------|--------|------|
| 받는 사람 | 근로자 | 근로소득자 |
| 받는 돈 | 임금 | 근로소득 |
| 세금 떼는 것 | - | 원천징수 (지방세는 특별징수) |
| 4대보험 | 원천공제 | - |

### 직원 vs 임원
| 구분 | 직원 | 임원 |
|------|------|------|
| 계약 상대 | 회사 (고용계약) | 주총 (위임계약) |
| 받는 돈 | 임금 | 보수 |
| 노동법상 근로자 | ⭕ | ❌ |
| 세법상 근로소득자 | ⭕ | ⭕ (과세 위해) |
| 퇴직금 | 법정 의무 | **원래 없음** (회사 정관에 따라) |

> 임원 퇴직금이 없는 회사도 많음 (예: 두산그룹)
> 임원 보수가 큰 이유 중 하나가 퇴직금 부재

## 11. 법인과 자연인

### 사람의 구분
- **자연인**: 태어나면서 사람 → 출생신고(주민센터) → 주민등록번호
- **법인**: 법에 의해 인정된 사람 → 출생신고(법원) → 법인등록번호

### 번호 체계
| 번호 | 발급 의미 | 자릿수 |
|------|----------|--------|
| 주민등록번호 | 출생신고 | 13자리 |
| 법인등록번호 | 법인 출생신고 | 13자리 (앞 6+뒤 7) |
| 사업자등록번호 | 사업 시작(세무서 등재) | 10자리 |
| 4대보험 관리번호 | 사업자등록 시 부여 | 11자리 |

### 회사의 3대 기관 (국가 구조 모방)
| 회사 | 국가 |
|------|------|
| 주주총회 | 국회 |
| 이사회 (대표이사) | 행정부 (대통령) |
| 감사 | 감사원 |

---

## 12. 세금의 구분

### 과세주체별
- **국세** → 세무서 (국세청)
- **지방세** → 지자체

### 결정방식별 (중요)
| 구분 | 신고납부세목 | 정부부과세목 |
|------|-------------|-------------|
| 다른 표현 | 자기결정 | 정부결정 |
| 결정주체 | 납세자 본인 | 정부 |
| 해당 세목 | 소득세, 법인세, 부가세 등 대부분 | **상속세, 증여세, 종부세** (3가지만) |
| 통제방식 | 사후 세무조사 | 사전 결정 |

> 결정 = 과세표준과 세액을 정하는 것

---

## 13. 신고 후 수정 (중요)

### 신고기한 기준
- **신고기한 내**: 자기 세액 결정 가능
- **신고기한 후 신고** = **기한 후 신고** → 결정력 없음 (세무서가 결정)

### 수정신고 vs 경정청구
| 구분 | 수정신고 | 경정청구 |
|------|---------|----------|
| 일상 용어 | 증액수정신고 | 감액수정신고 |
| 방향 | 세액 ↑ (더 내는 것) | 세액 ↓ (돌려받는 것) |
| 효력 | **자동 경정** | 청구 효력만 |
| 결정주체 | 납세자 본인 | 세무서가 결정 |
| 국가 손해 | 없음 (자동 인정) | 있음 (검토 필요) |

> **경정기관 = 세무서**
> 4대보험도 동일 용어 사용

### 예시
- 5/31 신고: 1억 → 8월에 1.5억으로 증액 = **수정신고** (자동 경정)
- 5/31 신고: 1억 → 8월에 0.5억으로 감액 = **경정청구** (세무서 승인 필요)

## 14. 원천징수의 법적 성격

### 이론적 배경
1. **공무수탁사인설**: 원천징수의무자가 국가 일을 대신해주는 민간인
2. **채무인수설**: 원천납세의무자(근로자)의 채무를 회사가 인수
   - 면책적 채무인수 / 병존적 채무인수 중 정설 없음

### 당사자 관계
| 용어 | 의미 |
|------|------|
| **원천징수의무자** | 회사 (소득 지급자) |
| **원천납세의무자** | 근로자/소득 귀속자 (원래 세금 낼 사람) |

---

## 15. 완납적 vs 예납적 원천징수

| 구분 | 완납적 원천징수 | 예납적 원천징수 |
|------|----------------|----------------|
| 의미 | 원천징수로 납세의무 종결 | 미리 떼고 추후 정산 |
| 대표 사례 | **복권 당첨금** | **근로소득** (연말정산) |

> 근로소득은 종합소득 → 원래 5월 종합소득세 신고 대상
> 회사가 대신해주는 것이 **연말정산**
> 근로소득 외 다른 소득이 있으면 별도 5월 신고 필요

## 16. 국가와 원천납세의무자의 관계

### 원칙
- 국세청은 **회사(원천징수의무자)에게 책임 추궁**이 원칙
- 직접 근로자에게 책임 추궁 불가

### 예외: 원천납세의무자에 대한 경정 (직접 조사 가능 사례)
1. **퇴사자 추징 시**: 연말정산 부당공제 등 적발 후 이미 퇴사한 경우
2. **위조 증빙 제출**: 가짜 기부금 영수증 등 (실무상 잘 안 받아들여짐)
3. 원천징수의무자가 부당공제 여부 확인이 어려운 경우

### 근로자의 직접 신청
- 과거: 회사만 연말정산 수정/경정 가능
- **현재: 근로자 본인도 수정신고/경정청구 가능** (법 개정)
- 환급도 근로자에게 직접 가능

## 17. 원천징수 대상자 (개인도 의무 발생)

### 핵심 포인트
- 원천징수의무자 = **사업자만이 아님**
- **거주자(개인)도 원천징수의무 발생** 가능

### 예시: 가족 간 차입금 이자
- 부모/형제에게 집 살 돈 빌리고 이자 지급 시
- **법정이자율: 4.6%**
- 원칙적으로 원천징수 대상 (실무상은 그냥 주는 경우 많음)

 # 원천징수실무 교육 정리 (3회차)

## 18. 원천징수의무자의 범위 (재정리)

### 핵심
- 원천징수 = 사업자만 하는 것이 아님
- **개인(거주자)도 원천징수 의무 발생**
- 단, **계산서/세금계산서 발행은 사업자만 가능**

> 예시: 가족에게 이자 지급 시 신고하면 원천징수 의무 발생

---

## 19. 원천징수 승계 (사업장 합병/폐업 시)

### 외환은행-하나은행 합병 사례
- 12월 31일자 합병 → 1월 1일부터 하나은행
- 외환은행 직원의 연말정산:
  - 12월 31일까지 외환은행 재직 → **외환은행이 연말정산 수행**
  - 환급액 발생 시 → 1월부터는 하나은행으로 직원 이동
  - **합병으로 사라진 회사의 환급액을 합병 후 회사가 승계**

### 서식 위치

## 30. 가산세 체계 정리 ⭐

### 일반 세금 가산세 (3종 세트)
| 가산세 | 계산식 |
|--------|--------|

### 지급명세서 가산세

|------|--------|----------|----------|

| **간이지급명세서** (사업·기타 매월) | 0.25% | 0.125% | **1개월 이내** |

- 중소기업: **5천만원**

## 31. 원천징수 납부지연 가산세 ⭐

### 옛 규정
- 잘못되면 무조건 **10%** → 너무 가혹

### 현행 규정
원천징수 납부지연 가산세 = MIN(A, B) (한도 10%)
- **A** = 본세 × **3%** + 본세 × 미납일수 × 22/100,000
- **B** = 본세 × **10%** (한도)

> 약 **319일** 지점에서 한도 10% 도달
> → 그 이상 지연 시 더 늘어나지 않음
> → 작년 연말정산 수정신고는 12월 안에 처리 권장

### 핵심 용어
- **49페이지 제목**: "원천징수 등 납부지연 가산세" → **"신고"라는 말 없음**
- → 원천세는 납부만 하면 가산세 없음을 의미

## 34. 근로소득 구조

## 52. 재무제표 기본 (참고)

### 재산 vs 소득
| 구분 | 재산 | 소득 |
|------|------|------|
| 보고서 | 재무상태표 (B/S, 옛 대차대조표) | 손익계산서 (I/S, ≠P/L 일본식) |
| 시점 | **특정 시점** (Stock) | **기간** (Flow) |
| 기준일 | 12/31 | 1/1~12/31 |

### 손익계산서 구조
```
매출액
 - 매출원가
 = 매출총이익
 - 판관비 (판매비와 관리비)
 = 영업이익  ← 회사 본업
 + 영업외수익
 - 영업외비용
 = 법인세차감전 순이익
 - 법인세
 = 당기순이익  ← 주주 몫 (배당 재원)
```

## 53. 성과급(인센티브) 핵심 정리 ⭐

### 귀속시기 ⭐
- **개인별 지급액이 확정되는 연도** (내가 얼마 받을지 아는 시점)
- 예: 2025년 성과평가 → 2026년 4월 지급 → **2026년 성과급**

### 임금성 판단 (노동법)
- **성과급(PS, Profit Sharing) = 임금 ❌**
- 이유: 주주 몫인 당기순이익을 직원과 공유하는 것
- 일의 대가가 아닌 **경영 파트너로서의 이익 공유**

### 임금성에 대한 입장 차이
| 구분 | 임금성 선호 |
|------|------------|
| **근로자 (순수)** | 임금 ⭕ (퇴직금·수당 산정 기준) |
| **회사** | 임금 ❌ (퇴직금·수당 절감) |

### 세법 핵심 - 비과세/과세 입장 차이
| 구분 | 비과세 선호 |
|------|------------|
| 근로자 | 비과세 ⭕ |
| 회사 | 비과세 ⭕ |
> 세법에서는 양측 의견 일치, 임금성에서는 정반대

### 종류
- **PI** (Performance Incentive): 일한 성과에 따른 인센티브
- **PS** (Profit Sharing): 이익 공유

 # 원천징수실무 교육 정리 (6회차)

## 61. 월정액 급여 판단 시 식대 처리 ⭐

### 핵심 결론
**식대는 월정액 급여에 포함된다** (비과세지만 포함)

### 이유
- 월정액 급여 = "매달 고정적으로 받는 금액"
- 식대는 매달 받는 고정 항목이므로 포함

### 판단 공식
```
월정액 급여 = 매월 급여 총액
            - 상여 등 부정기 급여 (매달 안 받는 것)
            - 실비변상적 비과세
            - 복리후생적 비과세
```
→ **식대는 빠지지 않음 = 포함**

> 2021년 비과세 재편 후 일반 비과세에 식대 등이 들어가면서 명확히 포함됨

## 63. 직급보조비 (116p) ⭐

### 일반 사기업
| 항목 | 과세 여부 |
|------|----------|
| **직책수당** | 과세 |
| **판공비** (법인카드, 임원) | 비과세 |

### 공무원 직급보조비
- 두 가지 성격 혼재 (직책수당 + 판공비)
- 건강보험공단 vs 공무원집단 소송 발생
  - 건강보험공단: "직책수당 성격이다"
  - 공무원집단: "판공비 성격이다"
- 공무원 집단 승소 (지면 국가가 공보료 환급 부담)
- **2014년 법 개정** → 2015년부터 **공무원 직급보조비 전액 과세**

## 64. 식대 비과세 (116~118p) ⭐⭐

### 기본 한도
- **월 20만원** 비과세 (옛 10만원에서 상향)

### 지급 형태별 처리
| 지급 형태 | 비과세 여부 |
|----------|------------|
| **현물 식사 제공** (구내식당 직접 운영) | **금액 제한 없이 전액 비과세** |
| **현금 지급** (식대) | 월 20만원 비과세 |
| **식권 지급** (현물 간주) | 전액 비과세 |
| **식권 + 현금 환전 가능** | 현금으로 보아 **월 20만원** 한도 |

### 외부 식당 정산 방법 (118p) ⭐
| 정산 방식 | 처리 |
|----------|------|
| **식권으로 정산** | 현물로 인정 → 전액 비과세 |
| **법인카드 1인당 결제** | 현금 간주 → **월 20만원 한도** |

### 식대 비과세 요건
1. 통상적으로 급여에 포함되지 않을 것
2. 음식물 제공 여부에 따라 급여에 차등이 없을 것
3. 사용자가 무상 또는 저가 제공

### 구내식당 부가세 면세 이유
- 구내식당은 면세 (대상이 직원으로 한정)
- 외부인에게 팔면 면세 깨짐 → 매출 봉쇄
- → 서울대 학생식당이 택시기사에게 안 파는 이유

## 65. 통근수당·자가운전보조금 (119~120p) ⭐

### 통근 관련
| 항목 | 처리 |
|------|------|
| 회사 통근버스 운영 | **근로소득 과세제외** |
| 출퇴근 교통비 현금 지급 | **과세** |
| 업무 출장비 | 비과세 |
| 포상 성격 회의 출장비 | **근로소득 과세** |

### 자가운전보조금 (120p)
- **월 20만원** 비과세 (옛 10만원에서 상향)

### 비과세 요건
1. 종업원 **소유** 또는 본인 **명의 임차** 차량 (옛 소유만 → 임차 추가)
2. 종업원이 **직접 운전**
3. **시내 출장 등 업무 수행에 이용**
4. 월 20만원 이내

### 실무 주의사항 ⭐
- 자가운전보조금 = **시내 출장비 대신**
- **출장비 별도 정산 시 비과세 적용 불가**
- **건강보험공단이 적극 추징**:
  - **내근직 직원에게 자가운전보조금 지급 → 무조건 문제 삼음**
- 대응 방법: **출장 기록** 조금이라도 만들어 놓을 것
- 출근 목적은 무관, **낮 시간 시내 출장** 목적이어야 함

### 현실적 한계
- 비과세 한도 20만원은 유가 800~900원 시절 만들어진 규정
- 현재 유가 대비 부족
- "시내" vs "시외" 구분 모호함

## 66. 차량 구입 vs 리스 vs 렌트 (실무 팁)

### 결론: **현금 여유 있으면 구입**

### 비교
| 구분 | 단점 |
|------|------|
| **구입** | 초기 자금 소요 (예: 3,000만원) |
| **리스** | 차량 법적 명의가 리스회사 → 등록세 2번 부담 (구입 + 인수 시), 리스 이자 부담 (전체 차값 기준), 약 3,500만원 |
| **렌트** | 렌트회사가 보험 가입 → **보험료에도 부가세 부담** (직접 가입 시 부가세 없음), 약 3,300만원 |

### 리스/렌트의 비용 처리는 환상
- "리스/렌트가 비용 처리에 유리"는 잘못된 인식
- **차량 구입도 감가상각으로 비용 처리 가능**
- 리스/렌트는 **더 많은 현금을 쓰는 것** → 비용 처리 위해 현금 더 쓰는 모순

### 렌트가 유리한 경우
- 직원에게 차량 운행을 맡길 때
- 사고 시 할증 위험 회피 목적
- 예: 사장 차량을 직원 3명이 운행하는 경우
